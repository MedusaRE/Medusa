about the project

Medusa is a project to create a cross platform, free, and general purpose tool for software and hardware research, like reverse engineering, analysis, emulation, development, debugging, and other similar tasks.

All of the code written specifically for the Medusa project is licensed under the GPL, specifically version 2 of the GNU General Public License.

Sub-projects / components of the project include:

  • libmedusa - a project to create a general purpose library & API for controlling emulated, and real-silicon machines.
  • libxParse - a project to create an executable parsing library/tool for general use.
